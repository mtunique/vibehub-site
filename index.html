How It Works

1

Install & Launch

Join the TestFlight beta or download from GitHub. VibeHub installs hook scripts into ~/.claude/hooks/ automatically.

2

Start Coding

Use Claude Code or OpenCode as usual. The hooks send session events to VibeHub via a local Unix socket.

3

Stay in the Flow

The Dynamic Island shows live status. Approve permissions, see results, and switch contexts — all without leaving your editor.
