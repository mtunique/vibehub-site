Vibe Hub

AI 编程助手的灵动岛。在 Mac 刘海或菜单栏实时监控 Claude Code 和 OpenCode 会话。

下载 Mac 版 GitHub

最新版本：v1.3.1 · macOS 15+ · 历史版本

实时会话追踪

一眼掌握所有 Claude Code 和 OpenCode 会话——本地和远程，附带实时状态指示。

工具调用可视化

实时查看每一次工具调用——文件编辑、命令执行、MCP 工具——并可直接批准或拒绝。

菜单栏模式

没有刘海屏？用螃蟹动画图标在任何 Mac 上显示实时状态。

远程 SSH 监控

添加远程服务器并通过 SSH 监控 AI 会话——点击眼睛图标自动切换到对应终端标签。

引导式配置

首次启动选择显示模式。自动模式根据屏幕类型选择灵动岛或菜单栏。

灵活配置

显示模式、通知提示音、多显示器支持、远程主机管理。

功能特性

🌎

灵动岛 UI

从 MacBook 刘海处流畅展开，显示会话详情，空闲时自动收起。

⚙

菜单栏模式

适用于所有 Mac，包括无刘海屏型号。动态螃蟹图标实时显示工作状态。

✨

自动适配模式

自动选择最适合当前屏幕的 UI——有刘海用灵动岛，否则用菜单栏。

⚡

实时工具追踪

实时观看 AI 助手的每一次文件编辑、命令执行和工具调用。

🔒

权限审批

直接在悬浮窗中批准或拒绝工具执行——无需切换终端窗口。

📡

原生 SSH 隧道

内置 libssh 反向 TCP 转发，无需额外进程。监控任意远程服务器上的 AI 会话。

📺
新功能

智能标签切换

点击眼睛图标，即使多个 SSH 连接指向同一台服务器，也能精准切换到对应的终端标签。

💬

对话历史

完整的对话历史，支持 Markdown 渲染。随时回顾 AI 助手说了什么、做了什么。

🔌

OpenCode 支持

同时支持 Claude Code 和 OpenCode CLI，首次启动自动安装 Hook 脚本。

工作原理

1

下载并启动

下载 DMG 并将 Vibe Hub 拖入应用程序文件夹。首次启动会自动安装 Hook 脚本到 ~/.claude/hooks/。

2

正常编程

像往常一样使用 Claude Code 或 OpenCode。Hook 脚本通过本地 Unix Socket 将会话事件发送给 Vibe Hub。

3

保持专注

实时状态、权限审批、任务完成通知——全部显示在刘海或菜单栏，无需切换窗口。

系统要求

✓ macOS 15.0+
✓ Claude Code 或 OpenCode CLI
✓ 任意 Mac（针对 MacBook 刘海屏优化）
